# react-redux-typescript

Typescript 3.0

## Overview

This is a framework for reactJS with typescript.

## Installation

> `git clone https://github.com/Nyura95/react-typescript.git`

> `cd ./react-typescript`

> `npm install`

## Development mode

> `npm run start`

## build

> `npm run build`

## run a server in production

> `npm run serve`

## lint your code

> `npm run tslint`

Enjoy !
